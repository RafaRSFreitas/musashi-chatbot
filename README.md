# Miyamoto Musashi Chatbot

A conversational AI chatbot that simulates dialogue with the legendary Japanese swordsman Miyamoto Musashi.

## Project Overview
- **Course:** Software Engineering - Bournemouth University
- **Technology:** React + Express
- **Historical Figure:** Miyamoto Musashi (1584-1645)
